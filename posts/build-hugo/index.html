Build Hugo

December 9, 2019

我是在ubuntu上搭建hugo博客的

1.首先，安装hugo。

sudo apt install hugo

hugo new site newblog

cd newblog
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml
# 或者，编辑config.toml，在最后加入
theme = "ananke"

hugo new posts/first-post.md

hugo server -D
hugo

2.将hugo部署到github pages

git config --global user.name "yourname"
git config --global user.email "youremail"

git config user.name
git config user.email

ssh-keygen -t rsa -C "youremail"

ssh-keygen -t rsa -C "youremail"

首先，创建两个仓库，分别名为your-blog和username.github.io。 在将your-blog的仓库clone到本地，

git clone git@github.com:username/username/your-blog.git
cd your-blog

rm -rf public

git submodule add -b master git@github.com:<USERNAME>/<USERNAME>.github.io.git public

在yourblog目录下新建deploy.sh，内容如下：

#!/bin/sh

# If a command fails then the deploy stops
set -e

printf "\033[0;32mDeploying updates to GitHub...\033[0m\n"

# Build the project.
hugo # if using a theme, replace with `hugo -t <YOURTHEME>`

# Go To Public folder
cd public

# Add changes to git.
git add .

# Commit changes.
msg="rebuilding site $(date)"
if [ -n "$*" ]; then
	msg="$*"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master

chmod +x deploy.sh
./deploy.sh "commit 1"
